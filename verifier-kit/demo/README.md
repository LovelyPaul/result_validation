# demo — 발표·시연 자료

verifier-kit을 스터디에서 시연하기 위한 자료.

| 파일 | 용도 |
|---|---|
| `PRESENTATION.md` | 발표 대본 (5막 구성·Q&A 대비·타이밍) |
| `break-the-gate.sh` | 라이브 클라이맥스 데모 — "검수기가 약해지면 CI가 막는다" |

## 라이브 데모 실행

```bash
# repo 루트 또는 어디서든 — 스크립트가 verifier-kit/ 위치를 스스로 찾는다
bash verifier-kit/demo/break-the-gate.sh
```

각 단계 사이 `⏎`(엔터)로 진행한다. 흐름:

1. 지금 게이트 통과 (세 에이전트 거부율 100%)
2. 매설 노트를 정상으로 바꿔 검수기 약화
3. 게이트 재실행 → **거부율 100%→75%, exit 1로 빌드 차단**
4. 원복

**부작용 0**: 매설 노트는 자동 원복된다(중간에 Ctrl-C 해도 `trap`으로 복구). 반복 실행 안전.

## 데모 전 체크

- 터미널 폰트 크게(18pt+). 게이트 실행은 ~0.25초라 대기 없음.
- 사전 자료 탭: 해부 아티팩트 · 구축 아티팩트 · GitHub Actions(초록 배지).
- 상세는 `PRESENTATION.md`.
